The display Property

display: none:

This text is hidden and will not appear on the page.

Notice that nothing is shown above because the paragraph with class "ex1" is set to display: none;

display: inline:

Text 1 Text 2 Text 3

The inline display keeps elements on the same line like regular text.

display: block:

Text 1 Text 2 Text 3

Block display makes elements take up the full width available, starting on a new line.

display: inline-block:

Text 1 Text 2 Text 3

Inline-block elements are like inline elements but respect width and height properties.

display: flex:

Text 1 Text 2 Text 3

Flex display arranges children in a flexible way, allowing alignment and spacing control.

display: grid:

Text 1 Text 2 Text 3
